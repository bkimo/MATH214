H(s) = s/(s²+π²)² → split as F(s)·G(s) → h(t) = t·sin(πt) / (2π)
STEP 1 — FACTOR
Split: F(s) = s/(s²+π²) and G(s) = 1/(s²+π²)
STEP 2 — INVERT
f(t) = cos(πt)
g(t) = (1/π)sin(πt)
STEP 3 — PRODUCT-TO-SUM
cos(A)·sin(B) = ½[sin(A+B)−sin(A−B)]
A+B = πt is constant in τ — integral simplifies.
STEP 4 — RESULT
h(t) = t·sin(πt)/(2π)
Amplitude grows linearly — resonance shape.
closed form t·sin(πt)/(2π)
Notice the shape: h(t) is a sine wave whose amplitude envelope grows as ±t/(2π) (the dashed lines). This linearly growing amplitude is the hallmark of resonance — compare with Lecture 13 where the same t·sin shape appeared.
y″ + 3y′ + 2y = r(t), y(0)=y′(0)=0 | r(t) = 1 if 1<t<2, else 0
STEP 1 — LAPLACE
(s²+3s+2)Y = (e⁻ˢ−e⁻²ˢ)/s
STEP 2 — SOLVE
Y(s) = G(s)·(e⁻ˢ−e⁻²ˢ)
where G(s) = 1/[s(s+1)(s+2)]
STEP 3 — PARTIAL FRACTIONS
g(t) = ½ − e⁻ᵗ + ½e⁻²ᵗ
STEP 4 — SHIFT THEOREM
y(t) = g(t−1)u(t−1) − g(t−2)u(t−2)
0 < t < 1Pulse has not arrived — system is at rest. y(t) = 0 exactly.
1 < t < 2Pulse is on. System is driven. Response builds toward steady state ½.
t > 2 (full width)Pulse switches off. Two decaying exponentials interfere. y(t) → 0.
Key insightThe pulse ends before y reaches ½ — so the system partially charges then freely decays.
y(t) — system response
r(t) — pulse (scaled ×0.3)
The response is zero until t = 1, rises smoothly during the pulse, then decays after t = 2. Note the slight inflection as the pulse shuts off — the system has stored energy and releases it gradually.
